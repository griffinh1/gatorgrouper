Product Desrcription

Gator Grouper is a website that helps assist teachers when organzing students into groups. It organizes these groups based off of the students skill levels.

To use this tool one can sign up via multiple ways. After that you can then login to your account, answer the survey, then be placed into groups with people in your class.

This tool is designed to create fair and balanced groups based on a set of questions. Professors, teachers, students, and anyone in need of creating balanced groups for projects is encouraged to try this tool.
